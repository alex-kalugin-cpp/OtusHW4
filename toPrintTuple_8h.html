Detailed Description

Заголовочный файл с вспмогательными функциями печати условного IP адреса из std::tuple.

Данный файл содержит в себе шаблонные функции, реализующие печать условного IP адреса из std::tuple и проверку аргументов исходного tuple Адрес должен быть представлен в виде кортежа с одинаковыми типами
